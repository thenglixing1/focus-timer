# Focus Timer Extension

A Pomodoro-style focus timer Chrome extension with browser notifications.

·Features
- 25-minute focus sessions with 5-minute breaks
- Browser notifications when timer ends
- Customizable focus and break durations
- Session counter

·Installation
1. Download or clone this repository
2. Open Chrome → chrome://extensions
3. Enable Developer Mode
4. Click "Load unpacked" and select this folder

·Permissions Used
- storage: Save timer settings and session count
- notifications: Alert user when timer ends
- alarms: Run countdown in background
