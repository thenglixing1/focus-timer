This focus timer can help solvw problem of lack of focus when study or working
this focus timer is similar like the pomodoro technique, it is about 25 minit start 
working and rest 5 minit, help user can focus to complete they work. 
Besides that, I used chrome.storage to save the 
timer settings and session count across browser sessions, used
chrome.alarms for the background countdown logic, and  
chrome.notifications to alert the user when the timer ends.
